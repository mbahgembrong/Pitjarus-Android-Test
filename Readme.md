# Kotlin Android
Pitjarus Test 
App for android written in kotlin.

## Features
- [x] Kotlin
- [x] Dagger Hilt
- [x] Retrofit
- [x] Room
- [x] Google Map
- [x] MVVM
- [x] Camera X
- [x] Timber

## Requirements
- JDK 16


## Installation
- Clone this repository and import into Android Studio
    ```bash
    git clone https://github.com/mbahgembrong/Pitjarus-Android-Test.git
    ```
- Setup SDK for project
- Jalankan Project in Android
